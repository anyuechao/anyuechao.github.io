我们能在Clang上做些什么

LibClang

(1)优缺点

1.C API 来访问Clang的上层能力,如获取Tokens、遍历语法树、代码补全、获取诊断信息

2.API稳定,不受Clang源码更新影响

3.只有上层的语法树访问,不能获取到全部信息

(2)LibClang - 如何使用

1.使用原始C API
2.脚本语言:使用官方提供的python binding 或开源的node-js/rubybinding
3.Object-c:开源库ClangKit

(3)LibClang-Demo

优缺点

1.对语法书有完全的控制权

2.作为插件注入到编译流程中,可以影响build和决定编译过程

3.需要使用C++且对Clang源码熟悉
